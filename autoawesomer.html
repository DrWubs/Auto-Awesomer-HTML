Dr Wubs' Auto Auto Awesomer

This script that allows Turntable.fm users to vote every song that is played as Awesome automatically

Read Before Continuing:

Auto Awesomer will not work if you have any extensions that effect the normal function or interface of Turntable.fm including the extension Turntable X for Google Chrome.

How to get Auto Awesomer

Drag this link in to your bookmarks bar - Dr Wubs' Auto Awesomer

How to use Auto Awesomer

1. When in a room on TT.fm, click the bookmark
2. The text "Dr. Wubs' Auto Awesome" should appear in the top left section of Turntable.fm
3. Each time a song plays you will automatically vote the song Awesome
4. The script will never turn off unless you exit a room or manually turn it off
5. You can turn off the script manually by clicking the red text "Turn off"

Features:

    Automatically Awesomes all song automatically

    Can be manually turned off

    Allows the ability to Lame songs

Not working?

1. In Google Chrome web browser Right Click the Bookmarklet bar

2. Click "Add Page"
3. Name it whatever you want
4. Right Click this link and click "Copy Link Address"
5.Paste the link address into the URL address bar
6. Click "SAVE"

Note: It is activated and deactivated the same way it was originally.
